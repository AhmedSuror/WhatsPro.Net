# API Reference

- `WhatsProClient.Auth` -> Login, GetProfile, UpdateProfile, ChangePassword
- `WhatsProClient.Dashboard` -> GetDashboard, GetTopNumbers, GetSendChart
- `WhatsProClient.Clients` -> List, Create, Get, Update, Delete, ImportFromExcel
- `WhatsProClient.Groups` -> List, GetAll, Create, Get, Update, Delete, TransferClients, DeleteClients
- `WhatsProClient.Sessions` -> List, Connect, Disconnect, Get, ChangeName, SetWebhook
- `WhatsProClient.Messages` -> List, Get, Delete, Send, SendNonEncrypted, UploadDocument
